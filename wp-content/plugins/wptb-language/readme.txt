== Description ==

With this plugin you can easily **switch the WordPress language**.<br />
It will download and install the language files (.po and .mo) from the WordPress Repository if they are not installed with WordPress.<br />
After that it will activate the selected language.

== Installation ==

Installation Option 1:
-----------------------
1. Download the file (wptblanguage.zip) to your computer
2. Login to your Wordpress Admin Area
3. Navigate to Plugins - Install Plugins - Add New - Upload
4. Upload the file, you saved on step 1. and when asked activate the plugin

Installation Option 2:
-----------------------
1. Login to your Wordpress Admin Area
2. Navigate to Plugins - Install Plugins
3. Search for "WPTB Language", in the search result find the exact title and click "Install"
4. When the installation finish click the "Activate Plugin" link in the installation report page or in the plugins list
